## Calculator Web App

A simple and interactive calculator built using HTML, CSS, and JavaScript.
This project was created as a practice exercise to strengthen fundamentals of frontend development and JavaScript logic.

![interface](/interface.png)

---

## Features

# Perform basic arithmetic operations:
- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- Decimal number support
- AC (All Clear) button to reset calculations
- DEL button to remove the last entered value
- Clean, minimal, and beginner-friendly UI
- Responsive button layout

--- 

## Tech Stack

HTML – Structure and layout

CSS – Styling and UI design

JavaScript – Calculator logic and event handling

---

## Important Note

- This calculator uses JavaScript’s eval() function to evaluate expressions.
- While suitable for learning and small projects, eval() is not recommended for production-level applications due to security concerns.
